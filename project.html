Project

iOS Travel App: One Journey

  • Developed a full stack iOS travel helper app that provided landmark recognition, real-time image translation, and popular poi recommendation. Enabled users' data persistence with Google Cloud Platform.
  • Collected more than 80,000+ landmark pictures by using web-crawling techniques, trained landmark classification model with these data on Google AutoML, increased classification accuracy from 82% to 93%.
  • Designed the recommendation of popular POI around through Google Map API and implemented automatic image recognition and translation through Google OCR and translate API.
  • Initiated an internal UI library based on UIKit to facilitate the development of compatibility for different devices.
